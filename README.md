# Real-Time-Data-Acquisition-and-Predictive-Thermal-Mitigation-Framework-for-BMS
Real-Time ESP32 based BMS that detects thermal imbalance between battery branches and applies staged protection :current throttling, branch isolation, and active cooling.
This mini-project focuses on real-time monitoring and early detection of thermal imbalance in a lithium-ion battery pack. The main goal is to identify uneven heating between parallel battery branches at an early stage and prevent overheating conditions before they develop into thermal runaway, improving overall safety and reliability. The system is built around an ESP32 microcontroller that continuously collects key electrical and thermal parameters such as voltage, current, and temperature from a dual 3S series battery pack consisting of two independent branches. Instead of relying only on fixed temperature thresholds, the design also analyzes the rate of temperature rise (dT/dt), which helps detect abnormal heating trends earlier than conventional methods.
For testing and validation, one branch is deliberately made thermally stressed by adding insulation to trap heat, while a DC motor is used as the load to create realistic operating conditions. This controlled imbalance helps evaluate how effectively the system can identify and respond to unequal thermal behavior across branches.
When a mismatch in thermal behavior is detected, the system responds in a staged manner:
It reduces the current in the affected branch using MOSFET-based PWM control, allowing that branch to cool while the other branch continues supplying power.
If the condition does not improve, the system isolates the faulty branch completely and continues operation in a reduced but safe single-branch (3S) mode.
As a final protective measure, a cooling fan is activated if internal enclosure temperature continues to rise, ensuring system-level thermal stability.
All monitored data is transmitted to a cloud-based dashboard for real time visualization and tracking. 
Overall, this approach enhances battery safety by combining real-time sensing, early thermal trend detection, and staged protective control  resulting in improved operational reliability, longer battery life, and better energy sustainability.
